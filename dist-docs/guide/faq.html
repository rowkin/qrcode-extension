常见问题 ​

这里收集了用户在使用二维码生成器时经常遇到的问题和解答。

安装相关 ​

Q: 如何安装扩展？ ​

A: 有两种安装方式：

  1. 从 Chrome 网上应用店安装（推荐）：

    • 访问 Chrome 网上应用店
    • 搜索"二维码生成器"
    • 点击"添加至 Chrome"

  2. 开发者模式安装：

    • 访问 chrome://extensions/
    • 开启"开发者模式"
    • 点击"加载已解压的扩展程序"
    • 选择项目目录

Q: 安装后看不到扩展图标？ ​

A: 请检查以下几点：

  • 确认扩展已成功安装（在 chrome://extensions/ 中查看）
  • 确认扩展已启用（开关处于打开状态）
  • 点击浏览器工具栏的扩展图标（可能需要点击"拼图"图标查看所有扩展）

Q: 支持哪些浏览器？ ​

A: 支持以下浏览器：

  • Chrome 88 及以上版本
  • Edge 88 及以上版本
  • 其他基于 Chromium 的浏览器 88+

使用相关 ​

Q: 如何生成当前页面的二维码？ ​

A: 有两种方式：

  1. 点击浏览器工具栏中的扩展图标
  2. 在页面空白处右键，选择"生成二维码"

Q: 如何为选中的文本生成二维码？ ​

A:

  1. 在网页中选中需要转换的文本
  2. 右键点击选中的文本
  3. 在右键菜单中选择"生成二维码"

Q: 如何为链接生成二维码？ ​

A:

  1. 在网页中找到需要生成二维码的链接
  2. 右键点击该链接
  3. 在右键菜单中选择"生成二维码"

Q: 如何复制 URL？ ​

A: 在二维码面板中：

  1. 找到 URL 显示区域
  2. 点击"复制"按钮
  3. URL 已复制到剪贴板

您也可以点击 URL 文本区域快速选中完整 URL，然后手动复制。

Q: 如何下载二维码图片？ ​

A:

  1. 在二维码面板中，将鼠标悬停在二维码上方
  2. 点击出现的"下载"按钮
  3. 二维码图片自动下载到默认下载目录

Q: 如何关闭二维码面板？ ​

A: 有三种方式：

  1. 点击面板右上角的关闭按钮（×）
  2. 点击面板外的遮罩层
  3. 按键盘上的 ESC 键

功能相关 ​

Q: 支持哪些内容类型？ ​

A: 支持以下内容类型：

  • 网页 URL
  • 纯文本
  • 链接地址
  • 其他可编码的文本内容

Q: 二维码有内容长度限制吗？ ​

A: 是的，二维码有内容长度限制：

  • 不同纠错级别支持的内容长度不同
  • 过长的文本可能无法生成二维码
  • 建议将超长内容分段处理

Q: 支持自定义二维码样式吗？ ​

A: 目前不支持自定义样式，但这是我们的未来计划之一。未来版本可能会支持：

  • 自定义二维码颜色
  • 调整二维码大小
  • 选择二维码样式模板

Q: 支持历史记录吗？ ​

A: 目前不支持历史记录功能，但这是我们的未来计划之一。

问题排查 ​

Q: 二维码无法生成？ ​

A: 请尝试以下方法：

  1. 检查内容是否过长
  2. 确认浏览器版本是否支持（Chrome 88+）
  3. 尝试刷新页面后重试
  4. 检查扩展是否已启用
  5. 重新加载扩展

Q: 二维码无法扫描？ ​

A: 请检查以下几点：

  1. 确保二维码清晰可见
  2. 检查屏幕亮度是否足够
  3. 尝试下载二维码图片后扫描
  4. 确保扫描设备摄像头正常工作

Q: 右键菜单不显示？ ​

A: 请尝试以下方法：

  1. 确认扩展已正确安装
  2. 检查扩展是否已启用
  3. 尝试重新加载扩展
  4. 刷新当前网页
  5. 重启浏览器

Q: 扩展图标点击无反应？ ​

A: 请检查：

  1. 确认扩展已启用
  2. 检查当前页面是否支持（某些特殊页面可能不支持）
  3. 查看浏览器控制台是否有错误信息
  4. 尝试重新加载扩展

Q: 复制功能不工作？ ​

A: 请检查：

  1. 确认浏览器支持 Clipboard API
  2. 检查浏览器权限设置
  3. 尝试手动选择文本复制
  4. 查看浏览器控制台是否有错误

技术相关 ​

Q: 数据会上传到服务器吗？ ​

A: 不会。所有处理都在本地完成：

  • 二维码生成在浏览器本地完成
  • 不收集任何用户数据
  • 不上传任何信息到服务器
  • 完全保护用户隐私

Q: 扩展占用多少资源？ ​

A: 扩展非常轻量：

  • 安装包大小：约 100KB
  • 内存占用：极小
  • 不影响浏览器性能
  • 仅在需要时加载资源

Q: 支持离线使用吗？ ​

A: 完全支持离线使用：

  • 所有功能都在本地实现
  • 不需要网络连接
  • 可以离线生成二维码

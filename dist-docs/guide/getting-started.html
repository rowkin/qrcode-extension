快速开始 ​

本指南将帮助您快速安装和开始使用二维码生成器 Chrome 扩展。

安装扩展 ​

方式一：从 Chrome 网上应用店安装（推荐） ​

  1. 访问 Chrome 网上应用店
  2. 搜索"二维码生成器"或"QR Code Generator"
  3. 点击"添加至 Chrome"按钮
  4. 在确认对话框中点击"添加扩展程序"
  5. 安装完成后，扩展图标将出现在浏览器工具栏中

方式二：开发者模式安装 ​

如果您需要安装开发版本：

  1. 下载或克隆项目代码
  2. 打开 Chrome 浏览器，访问 chrome://extensions/
  3. 开启右上角的"开发者模式"
  4. 点击"加载已解压的扩展程序"
  5. 选择项目根目录
  6. 扩展安装完成

首次使用 ​

安装完成后，您可以通过以下方式开始使用：

方法一：通过扩展图标 ​

  1. 访问任意网页
  2. 点击浏览器工具栏中的扩展图标
  3. 自动生成当前页面的二维码

方法二：通过右键菜单 ​

  1. 在网页中选中文本或右键点击链接
  2. 在右键菜单中选择"生成二维码"
  3. 自动生成对应内容的二维码

验证安装 ​

安装成功后，您应该能够：

  • ✅ 在浏览器工具栏看到扩展图标
  • ✅ 右键菜单中出现"生成二维码"选项
  • ✅ 点击扩展图标能够生成二维码

使用指南 ​

详细的使用说明和操作技巧，帮助您更好地使用二维码生成器。

基本使用 ​

生成当前页面二维码 ​

这是最常用的功能：

  1. 访问任意网页
  2. 点击浏览器工具栏中的扩展图标
  3. 自动弹出二维码面板，显示当前页面的二维码
  4. 使用手机扫描二维码即可访问该页面

生成选中文本的二维码 ​

适用于需要将文本内容转换为二维码的场景：

  1. 在网页中选中需要转换的文本
  2. 右键点击选中的文本
  3. 在右键菜单中选择"生成二维码"
  4. 二维码面板显示，包含选中的文本内容

使用场景：

  • 分享长文本内容
  • 传输配置信息
  • 分享代码片段

生成链接的二维码 ​

快速为页面中的链接生成二维码：

  1. 在网页中找到需要生成二维码的链接
  2. 右键点击该链接
  3. 在右键菜单中选择"生成二维码"
  4. 二维码面板显示，包含该链接的 URL

使用场景：

  • 分享页面中的外部链接
  • 快速在移动设备上打开链接
  • 分享下载链接

高级操作 ​

复制 URL ​

在二维码面板中：

  1. 找到 URL 显示区域
  2. 点击"复制"按钮
  3. URL 已复制到剪贴板
  4. 显示复制成功提示

提示：您也可以点击 URL 文本区域快速选中完整 URL，然后手动复制。

下载二维码 ​

保存二维码图片到本地：

  1. 在二维码面板中，将鼠标悬停在二维码上方
  2. 点击出现的"下载"按钮
  3. 二维码图片自动下载
  4. 文件名格式：qrcode-YYYYMMDD-HHmmss.png

提示：下载的二维码图片可以用于打印、分享或存档。

关闭面板 ​

有多种方式关闭二维码面板：

  • 方式一：点击面板右上角的关闭按钮（×）
  • 方式二：点击面板外的遮罩层
  • 方式三：按键盘上的 ESC 键

提示：关闭面板后，可以随时再次打开生成新的二维码。

使用技巧 ​

技巧一：快速分享网页 ​

  1. 打开要分享的网页
  2. 点击扩展图标生成二维码
  3. 使用手机扫描二维码
  4. 或下载二维码图片发送给他人

技巧二：批量处理文本 ​

如果需要为多个文本生成二维码：

  1. 逐个选中文本并生成二维码
  2. 每次生成后下载保存
  3. 或使用复制功能记录 URL

技巧三：跨设备同步 ​

  1. 在电脑上浏览网页
  2. 生成二维码
  3. 用手机扫描
  4. 实现跨设备无缝访问

技巧四：保存重要链接 ​

  1. 为重要链接生成二维码
  2. 下载二维码图片
  3. 保存到本地或云端
  4. 需要时扫描访问

常见使用场景 ​

场景一：会议分享 ​

在会议中需要分享网页链接时：

  1. 打开要分享的网页
  2. 生成二维码并投影到屏幕上
  3. 参会人员扫描二维码访问

场景二：教学演示 ​

在教学演示中：

  1. 为教学资源生成二维码
  2. 学生扫描二维码获取资源
  3. 提高教学效率

场景三：文档传输 ​

需要将文本内容传输到手机：

  1. 在电脑上选中文本
  2. 生成二维码
  3. 手机扫描获取文本

场景四：快速访问 ​

在移动设备上快速访问电脑上的网页：

  1. 电脑上生成二维码
  2. 手机扫描
  3. 立即在手机上打开

注意事项 ​

内容长度限制 ​

  • 二维码有内容长度限制
  • 过长的文本可能无法生成二维码
  • 建议将超长内容分段处理

特殊字符处理 ​

  • 支持大部分特殊字符
  • 某些特殊字符可能影响二维码识别
  • 建议使用标准字符集

网络链接 ​

  • 确保链接可访问
  • 某些需要登录的页面可能无法直接访问
  • 建议使用公开可访问的链接

故障排除 ​

二维码无法生成 ​

  • 检查内容是否过长
  • 确认浏览器版本是否支持（Chrome 88+）
  • 尝试刷新页面后重试

二维码无法扫描 ​

  • 确保二维码清晰可见
  • 检查屏幕亮度是否足够
  • 尝试下载二维码图片后扫描

右键菜单不显示 ​

  • 确认扩展已正确安装
  • 检查扩展是否已启用
  • 尝试重新加载扩展

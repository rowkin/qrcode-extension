快速开始

安装扩展

方式一：从 Chrome 网上应用店安装（推荐）

  1. 访问 Chrome 网上应用店
  2. 搜索"二维码生成器"或"QR Code Generator"
  3. 点击"添加至 Chrome"按钮
  4. 在确认对话框中点击"添加扩展程序"
  5. 安装完成后，扩展图标将出现在浏览器工具栏中

方式二：开发者模式安装

  1. 下载或克隆项目代码
  2. 打开 Chrome 浏览器，访问 chrome://extensions/
  3. 开启右上角的"开发者模式"
  4. 点击"加载已解压的扩展程序"
  5. 选择项目根目录
  6. 扩展安装完成

首次使用

方法一：通过扩展图标

  1. 访问任意网页
  2. 点击浏览器工具栏中的扩展图标
  3. 自动生成当前页面的二维码

方法二：通过右键菜单

  1. 在网页中选中文本或右键点击链接
  2. 在右键菜单中选择"生成二维码"
  3. 自动生成对应内容的二维码

使用指南

基本使用

生成当前页面二维码

  1. 访问任意网页
  2. 点击浏览器工具栏中的扩展图标
  3. 自动弹出二维码面板，显示当前页面的二维码
  4. 使用手机扫描二维码即可访问该页面

生成选中文本的二维码

  1. 在网页中选中需要转换的文本
  2. 右键点击选中的文本
  3. 在右键菜单中选择"生成二维码"
  4. 二维码面板显示，包含选中的文本内容

生成链接的二维码

  1. 在网页中找到需要生成二维码的链接
  2. 右键点击该链接
  3. 在右键菜单中选择"生成二维码"
  4. 二维码面板显示，包含该链接的 URL

高级操作

复制 URL

  1. 在二维码面板中找到 URL 显示区域
  2. 点击"复制"按钮
  3. URL 已复制到剪贴板
  4. 显示复制成功提示

下载二维码

  1. 在二维码面板中，将鼠标悬停在二维码上方
  2. 点击出现的"下载"按钮
  3. 二维码图片自动下载
  4. 文件名格式：qrcode-YYYYMMDD-HHmmss.png

关闭面板

  • 方式一：点击面板右上角的关闭按钮（×）
  • 方式二：点击面板外的遮罩层
  • 方式三：按键盘上的 ESC 键

技术文档

技术架构

核心技术栈

  • Chrome Extension Manifest V3：使用最新的扩展规范
  • QRCode.js：二维码生成库
  • 原生 JavaScript：无框架依赖，轻量高效
  • Canvas API：二维码图片导出
  • i18n API：国际化支持

核心组件

1. Background Script (background.js)

职责：

  • 处理右键菜单创建和管理
  • 处理扩展消息传递
  • 管理扩展生命周期

2. Content Script (content.js)

职责：

  • 在网页中注入二维码面板
  • 处理用户交互（复制、下载、关闭）
  • 管理面板的显示和隐藏

3. Popup Script (popup.js)

职责：

  • 处理扩展图标点击
  • 与 Background Script 通信
  • 触发二维码生成

常见问题

安装相关

Q: 如何安装扩展？

A: 有两种安装方式：

  1. 从 Chrome 网上应用店安装（推荐）：访问 Chrome 网上应用店，搜索"二维码生成器"，点击"添加至 Chrome"
  2. 开发者模式安装：访问 chrome://extensions/，开启"开发者模式"，点击"加载已解压的扩展程序"，选择项目目录

Q: 支持哪些浏览器？

A: 支持以下浏览器：

  • Chrome 88 及以上版本
  • Edge 88 及以上版本
  • 其他基于 Chromium 的浏览器 88+

使用相关

Q: 如何生成当前页面的二维码？

A: 有两种方式：

  1. 点击浏览器工具栏中的扩展图标
  2. 在页面空白处右键，选择"生成二维码"

Q: 二维码无法生成？

A: 请尝试以下方法：

  • 检查内容是否过长
  • 确认浏览器版本是否支持（Chrome 88+）
  • 尝试刷新页面后重试
  • 检查扩展是否已启用
  • 重新加载扩展

技术相关

Q: 数据会上传到服务器吗？

A: 不会。所有处理都在本地完成：

  • 二维码生成在浏览器本地完成
  • 不收集任何用户数据
  • 不上传任何信息到服务器
  • 完全保护用户隐私

# 功能特性

二维码生成器提供了丰富而实用的功能，让二维码生成变得简单高效。

## 核心功能

### 1. 多种二维码生成方式

#### 通过扩展图标生成
- 点击浏览器工具栏中的扩展图标
- 自动生成当前页面的二维码
- 适用于快速分享当前浏览的网页

#### 通过右键菜单生成
- **生成选中文本的二维码**：选中任意文本，右键选择"生成二维码"
- **生成链接的二维码**：右键点击链接，选择"生成二维码"
- **生成当前页面的二维码**：在页面空白处右键，选择"生成二维码"

### 2. 用户友好界面

#### 浮动面板设计
- 居中显示的浮动面板，不遮挡页面内容
- 半透明遮罩背景，突出二维码显示
- 响应式布局，适配不同屏幕尺寸

#### 动画效果
- 平滑的打开/关闭动画
- 优雅的加载动画过渡
- 清晰的错误状态提示

### 3. 便捷操作

#### URL 管理
- **一键复制 URL**：点击复制按钮快速复制当前 URL
- **快速选择文本**：点击 URL 区域可快速选择完整文本
- **复制成功提示**：复制成功后显示视觉反馈

#### 二维码下载
- **一键下载**：点击下载按钮保存二维码图片
- **自动命名**：自动生成带时间戳的文件名
- **下载提示**：下载成功后显示通知
- **悬停显示**：鼠标悬停时显示下载按钮

#### 面板控制
- **关闭按钮**：点击关闭按钮关闭面板
- **遮罩关闭**：点击遮罩层关闭面板
- **ESC 快捷键**：按 ESC 键快速关闭
- **自动清理**：支持多次打开，自动清除旧面板

### 4. 多语言支持

- **中文**：完整的中文界面和提示
- **英文**：英文界面支持
- **日文**：日文界面支持
- **自动适配**：自动识别浏览器语言并切换

## 技术特性

### 高质量二维码生成
- 使用 QRCode.js 库生成高质量二维码
- 支持多种内容类型（URL、文本、链接）
- 清晰的二维码显示，易于扫描

### 性能优化
- 轻量级实现，不影响浏览器性能
- 快速响应，即时生成二维码
- 内存占用小，资源消耗低

### 安全可靠
- 基于 Chrome Extension Manifest V3
- 遵循 Chrome 扩展安全标准
- 本地处理，数据不上传

## 使用场景

### 场景一：分享网页
快速生成网页二维码，方便在移动设备上访问或分享给他人。

### 场景二：传输文本
将选中的文本生成二维码，快速传输到其他设备。

### 场景三：链接分享
为页面中的链接生成二维码，方便移动端访问。

### 场景四：跨设备访问
在电脑上浏览网页，生成二维码后用手机扫描访问。

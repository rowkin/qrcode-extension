## 为什么选择二维码生成器？

二维码生成器是一个专为 Chrome 浏览器设计的轻量级扩展，旨在帮助用户快速、便捷地生成二维码。无论是分享网页链接、传输文本信息，还是快速访问内容，都能轻松完成。

### 核心优势

- **简单易用**：无需复杂配置，安装即可使用
- **功能丰富**：支持多种生成方式和便捷操作
- **性能优秀**：轻量级实现，不影响浏览器性能
- **持续更新**：活跃的开发和维护，不断优化用户体验

## 快速体验

1. **安装扩展**：从 Chrome 网上应用店安装
2. **点击图标**：在任意网页点击扩展图标
3. **生成二维码**：自动生成当前页面的二维码

就是这么简单！
